Lavalamp for jQuery Lovers

A nifty animated menu effect — just 700 bytes minified. A jQuery port of the original Mootools version by Guillermo Rauch.

Compatible with jQuery 1.11.1, jQuery Easing 1.3

Lavalamp creates a smooth background-sliding highlight effect for navigation menus. The background element glides from one menu item to another, following your mouse with a customizable easing effect. It weighs just 700 bytes minified.

Installation

Step 1 — HTML

Use a semantically correct <ul> list for your navigation.

<ul class="lavaLamp">
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Work</a></li>
    <li><a href="#">Contact</a></li>
</ul>

Step 2 — CSS

The "sliding doors" technique: the .lavaLamp list has position: relative, and the background highlight element is absolutely positioned inside it.

ul.lavaLamp {
    position: relative;
    list-style: none;
    margin: 0;
    padding: 0;
}
ul.lavaLamp li {
    float: left;
    cursor: pointer;
}
ul.lavaLamp li.back {
    position: absolute;
    top: 0;
    z-index: 0;
}

Step 3 — JavaScript

Initialize lavaLamp on your navigation element. Optionally pass an easing function and speed.

$(function() {
    $(".lavaLamp").lavaLamp({
        fx: "backout",
        speed: 700
    });
});

The fx option accepts any jQuery Easing value. speed controls the animation duration in milliseconds.
