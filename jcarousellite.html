jCarouselLite

A super light-weight (~2 KB) jQuery plugin for creating carousels with images or any HTML content.

Version 1.1 — Sept 2014  ·  GitHub  ·  Compatible with jQuery 1.11.x, Mousewheel 3.1.x, Easing 1.3

Information

Why?

jCarouselLite is a jQuery plugin that carries you on a carousel ride filled with images and HTML content. Put simply, you can navigate images and/or HTML in a carousel-style widget. It is super light-weight, at about 2 KB, yet very flexible and customizable to fit most needs.

How?

It requires no special CSS file or class names. You just include the JS file, supply the HTML markup (a div enclosing a ul), and call the function. That's it — the carousel calculates dimensions automatically based on your items.

What's inside?

  • A single ~2 KB minified JavaScript file
  • No required CSS — the plugin auto-sizes based on item dimensions you provide
  • Optional support for the jQuery Easing and MouseWheel plugins
  • Horizontal and vertical scrolling, circular navigation, auto-scroll, and more

Installation

Step 1 — Include files

<!-- Required -->
<script type="text/javascript" src="path/to/jquery.js"></script>
<script type="text/javascript" src="path/to/jcarousellite.js"></script>
<!-- Optional -->
<script type="text/javascript" src="path/to/easing.js"></script>
<script type="text/javascript" src="path/to/mousewheel.js"></script>

Step 2 — HTML markup

<button class="prev">&laquo;</button>
<button class="next">&raquo;</button>
<div class="any-class">
    <ul>
        <li><img src="image/1.jpg" style="width:150px; height:118px;"></li>
        <li><img src="image/2.jpg" style="width:150px; height:118px;"></li>
        <li><img src="image/3.jpg" style="width:150px; height:118px;"></li>
        <li><img src="image/4.jpg" style="width:150px; height:118px;"></li>
        <li><img src="image/5.jpg" style="width:150px; height:118px;"></li>
    </ul>
</div>

Step 3 — Initialize the plugin

$(function() {
    $(".any-class").jCarouselLite({
        btnNext: ".next",
        btnPrev: ".prev"
    });
});

Documentation

All configuration options are passed as a plain object to jCarouselLite().

Option Type Default Description
btnNext Selector — jQuery selector for the "next" button.
btnPrev Selector — jQuery selector for the "previous" button.
btnGo Array — Array of selectors for custom navigation (e.g., tabbed interfaces).
mouseWheel Boolean false Enable mouse-wheel navigation (requires the mousewheel plugin).
auto Number null Milliseconds between auto-scrolls. null disables auto-scroll.
speed Number 200 Animation speed in ms. Set to 0 to remove the slide effect.
easing String null Easing effect name (requires the jQuery Easing plugin).
vertical Boolean false true for a vertical carousel, false for horizontal.
circular Boolean true Enable infinite/circular wrap-around navigation.
visible Number 3 Number of items visible at once. Decimals like 3.5 tease the next item.
start Number 0 Zero-based index of the starting item.
scroll Number 1 Number of items to scroll per button click.

Styling – Default & Custom

The plugin handles horizontal/vertical alignment and sizing based on the visible count. You must provide the dimensions of individual items in your HTML or CSS.

Custom styling example

#jcl-demo .carousel {
    border: 1px solid #bababa;
    border-radius: 10px;
    background-color: ghostwhite;
    float: left;
    padding-left: 10px;
}
#jcl-demo .carousel-ul-li-img {
    width: 150px;
    height: 118px;
    vertical-align: middle;
    margin: 10px 10px 10px 0;
}

Known Issues

  • IE 6.0 width issue — In some configurations, the carousel width may not be calculated correctly in Internet Explorer 6. Specifying explicit item widths in CSS resolves this.
  • Image dimensions required — The plugin relies on item dimensions to compute the carousel width. Always specify width and height on your <img> elements or via CSS, otherwise the carousel may render incorrectly.
  • Flicker on load — On slow connections, unstyled list items may briefly be visible before the plugin initialises. To prevent this, hide the carousel container in CSS and show it in your afterEnd callback, or initialise the plugin inside $(window).load() rather than $(document).ready().
